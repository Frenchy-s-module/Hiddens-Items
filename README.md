Hidden Items is a Foundry VTT module that allows GMs to hide items from character sheets and manage them privately. Items can be easily hidden and revealed, with a convenient interface for managing hidden items. Perfect for secret items, quest objects, or any items that should be temporarily hidden from players.

### Features
- Real-time popup synchronization with the actor sheet (including item creation, deletion, update)
- Green/red eye icons to indicate visibility
- Neutral, modern popup style (adapts to all game genres)
- Dedicated button to open the popup (never opens automatically)
- GM-only access for managing hidden items
- Compatible with Foundry VTT v12â€“v13 and multilingual
- Debug logging available via CONFIG.debug.hiddenItems
- CSS customization possible via variables
- Perfect for quest items, secrets, or temporary item hiding
- Works with all systems using standard item management

### Installation
1. Inside Foundry VTT, select the Game Modules tab in the Configuration and Setup menu
2. Click the Install Module button and enter the following URL: `https://github.com/Frenchy-s-module/Hiddens-Items/releases/download/V1.2.0/module.json`
3. Click Install and wait for installation to complete

### Usage
1. Open a character sheet
2. Click on the eye icon next to an item to hide/show it
3. Access hidden items through the dedicated tab "Hidden Items (GM Only)"
